## About ZJView

**ZJView** helps you analyze and extract insights from large JSON files directly in your browser â€” no server, no data sent.

### Features

- Built with **Zig + WebAssembly**
- Supports files up to **500MB**
- Extracts and displays:
  - File size
  - Total keys
  - Total objects and arrays
  - Maximum depth
- Search for specific key values
- Extract and view all keys
- Works entirely offline for full privacy
